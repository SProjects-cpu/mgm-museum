## Citation Format

This bibliography follows the **APA 7th Edition** citation style for consistency and academic rigor. All web sources were accessed and verified between January 2025 and January 2026.

## Usage Guidelines

When citing references from this bibliography in your project report:

1. **In-text citations:** Use author-date format: (Martin, 2017) or (W3C, 2023)
2. **Multiple authors:** Use "et al." for 3+ authors: (Turban et al., 2017)
3. **Web sources:** Include retrieval date if content may change
4. **Direct quotes:** Include page numbers when available
